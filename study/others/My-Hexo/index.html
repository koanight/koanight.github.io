搭建我的Hexo博客

git

安装

 apt/yum/dnf install git -y

nodejs

下载

wget https://nodejs.org/dist/v18.12.1/node-v18.12.1-linux-x64.tar.xz

解压

tar xf node-v18.12.1-linux-x64.tar.xz -C /opt

配置环境变量

echo "PATH=\$PATH:/opt/node-v18.12.1-linux-x64" >> /etc/profile
. /etc/profile

添加淘宝源

npm config set registry https://registry.npmmirror.com

hexo

安装脚手架

npm i -g hexo-cli

建站

cd /opt
hexo init hexo
npm i

参考文档修改_config.yml

配置 | Hexo

修改模板字段

vim scaffolds/post.md 添加以下内容
    categories:

自行切换主题并参考主题说明修改相应配置

command

指令 | Hexo

release

pm2

安装pm2

npm i -g pm2

编写脚本

hexo-deploy.js

//deploy
const { exec } = require('child_process')
exec('hexo g -d (-w 用户监控文件改动情况自动发布)',(error, stdout, stderr) => {
        if(error){
                console.log('exec error: ${error}')
                return
        }
        console.log('stdout: ${stdout}');
        console.log('stderr: ${stderr}');
})

hexo-run.js

//run
const { exec } = require('child_process')
exec('hexo s (-p 80 用户指定端口)',(error, stdout, stderr) => {
        if(error){
                console.log('exec error: ${error}')
                return
        }
        console.log('stdout: ${stdout}');
        console.log('stderr: ${stderr}');
})

启动后台任务

pm2 start hexo-deploy.js
pm2 start hexo-run.js

注：使用nginx发布时，不需要使用pm2启动hexo-run.js或者通过hexo s发布

nginx

安装nginx

apt/yum/dnf install nginx -y

添加路由配置

root /opt/hexo/public;

location / {
   index index.html;
}

启动nginx

systemctl start nginx

可以使用pm2后台控制hexo的构建，nginx配置路由及https证书

github page

生成密钥

ssh-keygen -t rsa -C github邮箱地址

cat id_rsa.pub

在github页码中配置密钥

安装插件

npm i hexo-deployer-git --save

修改配置

deploy:
  type: git
  repo: <repository url>  # ssh地址
  branch: <branch name>   # master

配置git信息

git config --global user.name "github用户名"
git config --global user.email "github邮箱地址"

构建发布

hexo clean
hexo g -d
